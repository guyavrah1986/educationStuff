2) Ways to prevent dead-lock ?
a) Lock ordering: Make sure all threads aquire ALL locks (in case there are several) in the same order.
b) Enable preemption of "lower priorty" threads in case they hold a lock that a "higher priority" thread needs.
c) Deadlock detection: The OS takes care of it while utilizing some data structures that records each thread that aquired a lock in some graph or other 
form. This way, it might be possible to "for-see" a potential deadlock situation and prevent it while it can.
